Structure for Day6

* Created a file notes.txt with touch command
* Added 1st line with =

echo "text" > notes.txt

* Added 2nd line with =

echo "text" >> notes.txt

* Added 3rd line =

echo "text" | tee -a notes.txt

and print

* Cat = to print the file data
* Head -n 2 notes.txt = to print first two line of file
* Tail -n 2 notes.txt = to print last 2 lines of file
