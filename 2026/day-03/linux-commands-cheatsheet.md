Process Management

S.No/Command/Usage

1	/top	/This command is used to monitor system processes such as CPU usage, RAM usage, system uptime, load average etc.

2	/htop	/It is an enhanced version of the top command for easy navigation and troubleshooting because of its beautiful UI.

3	/uptime	/Shows real-time load by telling since how long the machine is working.

4	/ps aux	/It is used to display all running processes on a Linux system. It is different from the “top” command in the way that the top command is continuously updating and “ps aux” is just a screenshot.

5	/kill pid	/This command is used to terminate a process using PID.

File System Commands

S.No/Command/Usage

1	/cp filename (cp source destination)	/This command is used to copy files and directories.

2	/mv filename (mv source destination)	/This command is used to move file names or rename them.

3	/head file.txt	/Shows the first 10 lines.

4	/tail	/To display the last 10 lines.

5	/chmod	/It is used to change file permissions.

Networking Commands

S.No/Command/Usage

1	/ping	/This command pings the source and checks whether it is working or not.

2	/ifconfig	/This command tells the IP address configuration.

3	/netstat -tulp /Shows listening hosts or ports where the source is deployed.

4	/curl	/It is used to test the API response
